# Electric Vehicle Data205 Capstone Project
# Author: Damir Kumukov
This project analyzes the growth of electric vehicle adoption in Maryland and compares it to the availability of charging infrastructure. The goal is to determine whether infrastructure is keeping pace with increasing EV demand across counties.
My research goal is to evaluate whether charging infrastructure in Maryland is expanding at a rate that matches EV adoption.
# Datasets Used
EV Registration Data (MDOT/MVA)
Monthly data from 2020-2025
County-level EV counts
Used for trend analysis
Alternative Fuel Stations (U.S Department of Energy)
Snapshot from February 2026
Includes station locations and charger types
Used to measure infrastructure availability
# Methods Used in This Project
Data Processing
Cleaning and grouping data by county
Merging datasets for comparison
Visualizations
Bar charts for county comparisons
Line charts for trends
Maps for geographic distribution
Scatter plots for relationships
# Models 
Linear Regression: EV Count ~ Charging Stations
Multilinear Regression: EV Count ~ DC Fast Charger + Level 2 Charger
